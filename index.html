What features do you think an ideal Discord BOT should have?

Oh boy! You found a bug in the BOT?

I can't say how thankful I am. I'll fix that as soon as possible.

See if it's already reported in GitHub Issues Page or in #bugs channel of Bastion BOT Official Discord Server. If it's not reported, either create a new issue in GitHub or report it in #bugs channel of Bastion BOT Official Discord Server

You will get 50 Bastion Currencies for every valid bug report.

Have any suggestions for the BOT?

Dude, I'll be glad to implement that into the BOT if it's doable.

You can send us your suggestions in respective channels in Bastion BOT Official Discord Server. Or you can do a PR with your suggestions in the Bastion BOT Github Repo.

You will get 50 Bastion Currencies if your suggestion is implemented in the BOT.
