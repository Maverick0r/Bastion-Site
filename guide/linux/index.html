Bastion Installation Guide - GNU/Linux

Want to host Bastion on a VPS?

If you want Bastion to server 24/7 without having to hosting it on your PC and want to keep it cheap, reliable and convenient as possible, you can install Bastion, using following guide, on a Digital Ocean Linux VPS using this link (using this link will be supporting Bastion and will give you $10 credit)

1. Download the Installer

Open Terminal, and type: 
wget https://github.com/snkrsnkampa/Bastion-Scripts/raw/master/BastionInstaller.sh && chmod +x BastionInstaller.sh
and press Enter.

2. Install Bastion

The installer you downloaded for your distro will automatically download and install everything you'll need to run Bastion BOT.
To start the installer, type: 
sudo ./BastionInstaller.sh
and press Enter.
Wait a while for Bastion installer to finish installing.
You should now have a new folder named Bastion in your HOME (~) directory.

3. Configuration

If you didn't set up BOT ID, BOT Token, Owner ID, Command Prefix, BOT Status & BOT Game during BOT Installation, read the Configuration section in F.A.Q.

4. Running Bastion BOT

Assuming you are still inside Bastion directory, in you Terminal, type:
sudo ./run.sh
